# docs : 문서 (문서 추가, 수정, 삭제)

# test : 테스트 (테스트 코드 추가, 수정, 삭제: 비즈니스 로직에 변경 없음)

# chore : 기타 변경사항 (빌드 스크립트 수정 등)

# post : 블로그 포스트 추가 (신규 포스트 작성 및 수정)

# rename : 파일 혹은 폴더명을 수정하거나 옮기는 작업만 하는 경우

# remove : 파일을 삭제하는 작업만 수행한 경우
